# MacroMind - Fitness Tracking Application

A comprehensive web-based fitness tracking application built with Java Servlets, JSP, and MySQL. MacroMind helps users monitor their workouts, track food intake, and analyze their fitness progress.

## üéØ Features

### Core Functionality
- **User Management**: Registration, login, and profile management
- **Workout Logging**: Track cardio and strength training exercises with automatic calorie calculations
- **Food Entry Tracking**: Log daily food intake with macronutrient breakdown (calories, protein, carbs, fat)
- **Workout History**: View, filter, and manage past workouts
- **Progress Dashboard**: Visualize fitness metrics, calorie balance, and progress over time
- **Calorie Balance**: Monitor daily calorie intake vs. burned calories

### Key Capabilities
- Automatic calorie burn calculation based on exercise type, duration, and user weight
- Daily fitness summary with aggregated metrics
- Personalized daily calorie intake recommendations based on user goals
- Exercise distribution analysis (cardio vs. strength training)
- Session-based authentication and authorization
